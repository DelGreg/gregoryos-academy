# Installation iPhone — GrégoryOS Academy V4

## Ce qui change dans la V4

- L'application ne doit être publiée qu'une seule fois.
- Chaque matin, le JSON reçu dans ChatGPT se colle directement dans l'onglet **Progrès**.
- Aucun commit GitHub quotidien.
- La progression, les XP, le vocabulaire et le briefing installé sont conservés sur l'iPhone.

## Publication unique avec GitHub Pages

1. Décompresser le ZIP dans l'app Fichiers.
2. Ouvrir le dépôt `DelGreg/gregoryos-academy` dans Safari.
3. Utiliser **Add file → Upload files**.
4. Sélectionner les six fichiers utiles :
   - index.html
   - briefing.json
   - manifest.json
   - sw.js
   - icon-180.png
   - icon-512.png
5. Valider avec **Commit changes**.
6. Ouvrir **Settings → Pages**.
7. Choisir **Deploy from a branch**, branche `main`, dossier `/ (root)`, puis **Save**.
8. Ouvrir l'adresse fournie par GitHub Pages dans Safari.
9. Partager → **Sur l'écran d'accueil** → Ajouter.

## Routine quotidienne

1. À 6 h 30, ouvrir le briefing JSON reçu dans ChatGPT.
2. Copier tout le JSON.
3. Ouvrir Academy.
4. Onglet **Progrès**.
5. Coller dans **Installer le briefing du jour**.
6. Appuyer sur **Installer**.

Temps : environ 30 secondes.

## Important

Le briefing de marché est une photographie au moment de sa génération, pas un flux de cours en temps réel.
