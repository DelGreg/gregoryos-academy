# GrégoryOS Academy V4

Application web installable sur iPhone, conçue pour un briefing quotidien approfondi.

Fonctions :
- actualités économiques détaillées et sourcées ;
- marchés et mini-graphiques ;
- banque, BNP Paribas et lien métier ;
- IA & Data ;
- anglais avec audio et répétition espacée ;
- Stade Rennais FC et actifs sportifs ;
- quiz, XP, badges, midi et soir ;
- mise à jour quotidienne locale par collage du JSON.

Commencer par `INSTALLATION-IPHONE.md`.
